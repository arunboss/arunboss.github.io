Usage

Once you install the source code as shared in Installation page, you get a simple command line interface to run your face recognition use case.

The basic construct of the Input folder is as follows:

  • Unknown Photos – To keep the batch of photos which are unclassified
  • Known Photos – To keep the batch of known photos. The classification of each photo is kept in a dfInputRaw.csv in root folder

The algorithm stacks up the Unknown Photos into what is called Encodes. If you want to understand this step in geek details, read Under the Hood. Then the algorithm traverses through the Known Photos & tags encodes which match within tolerance.
